## AudioPlayer – це React-компонент для відтворення музики, який дозволяє користувачеві:

- Відтворювати, ставити на паузу та перемикати треки (Next / Previous);

- Обирати будь-яку пісню зі списку та починати її програвання;

- Відображає поточну пісню;

- Підсвічує активну пісню у списку для зручності;

- Має стильні кнопки управління.

## Особливості:

- Використання хука useRef();

- Управління відтворенням через стан useState();

- Підсвічування активної пісні через CSS;

- Стильні кнопки з hover і active ефектами.


## Технології:

- React (useRef(), useState());

- CSS / Flexbox;

- JavaScript (ES6+).
